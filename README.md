This is the home of the Noodles community Foundry server and dedicated gaming servers.

The Foundry cluster includes deployments for ArgoCD, Grafana's LGTM stack, Prometheus metrics, and Foundry itself. It includes automated backups for the Foundry server to an S3 bucket for archival, and automatic Cloudflare updates.

The games cluster currently runs Enshrouded, Satisfactory, and Valheim, as well as the community Discord bot.
